## Pre-execution
1. Create a Hugging Face account and generate a free token: https://huggingface.co/settings/tokens
2. Open `docker-compose.yaml` and replace `HUGGINGFACE_KEY` with the generated token.
3. Make sure Docker Desktop (or Engine) is running.

## Starting the environment
```
docker-compose -p ragstack up --build -d
```

> Note that it's necessary to adjust the ElasticSearch hostname in Python modules if the container name is different from `elasticsearch`.

## Access
- Airflow: http://localhost:8080 (user: airflow / password: airflow)
- Streamlit: http://localhost:8501
- Grafana: http://localhost:3000 (user: admin / password: admin)
